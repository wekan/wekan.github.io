Login help

If you can not login, please email support@wekan.fi from your login email address. Or alternatively, try the tips below.

a) Android: At Play Store install Chrome and WeKan, with Android ⚙ Settings / Default Settings set Chrome as default browser, with Chrome login to mail.google.com, with WeKan click Login with Google. After that, WeKan starts fullscreen.

b) iOS: At App Store install Chrome, with Chrome login to mail.google.com, with Chrome at boards.wekan.team click Login with Google.

c) Ubuntu Touch: At OpenStore install WeKan, with Morph browser login to mail.google.com, with WeKan click Login with Google.

d) Windows: At MS Store install WeKan, with WeKan click Login with Google.

e) Desktop browsers based on Safari/Firefox/Chromium: Login at boards.wekan.team. Compatible Webbrowsers.

f) Password: Write email address and password, and click Sign In at bottom.

g) Your own WeKan server URL: Create your own app icon
